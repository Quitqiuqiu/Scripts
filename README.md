# 一些自用脚本
## 1、WordCloud
由txt生成词云图，支持中文，基于`wordcloud`库，使用哈工大停用词表。
## 2、Merge_txt_to_csv
将多个txt文件合成一个csv文件
## 3、GetSteamGameTime
获取Steam游戏时长，降序排列，过滤掉时长为0的游戏，保存为`steam_playtime.csv`
## 4、RSA
加密、解密 2 种模式，加密需要 2 个素数 p q 与公钥 e，解密需要模数 n 与私钥 d 。
## 5、RSA_Key_Generation
2 种功能：根据 p 和 q 及输入的 e 生成公钥和私钥，根据 e 和 n 推导 p 和 q。
